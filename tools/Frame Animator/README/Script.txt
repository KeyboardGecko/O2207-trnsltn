Команды для скриптов

SET_FRAME_OFFSET - установка смещения фрейма
SET_FRAME_OFFSET,Frames,Frame,OffsetX,OffsetY,[Inherit]
  Frames - индекс направления ( 0 - North East, 1 - East,  2 - South East, 3 - South West, 4 - West, 5 - North West )
  Frame - индекс фрейма в заданном направлении ( 0, 1, ... FramesCount-1  или  -1, -2, ... -FramesCount )
  OffsetX - смещение фрейма по X ( положительное или отрицательное целое число )
  OffsetY - смещение фрейма по Y ( положительное или отрицательное целое число )
  Inherit - флаг наследования смещения ( 0 - нет, 1 - да, по умолчанию - 0 )
  Примеры:
    SET_FRAME_OFFSET,0,3,2,2,1
    SET_FRAME_OFFSET,5,-1,0,-1

SET_FRAMES_OFFSET - установка смещения направления  
SET_FRAMES_OFFSET,Frames,OffsetX,OffsetY
  Frames - индекс направления ( 0 - North East, 1 - East,  2 - South East, 3 - South West, 4 - West, 5 - North West )
  OffsetX - смещение направления по X ( положительное или отрицательное целое число )
  OffsetY - смещение направления по Y ( положительное или отрицательное целое число )
  Примеры:
    SET_FRAMES_OFFSET,3,1,-1

ADD_FRAMES - добавление новых фреймов
ADD_FRAMES,Frames,Image
  Frames - индекс направления ( 0 - North East, 1 - East,  2 - South East, 3 - South West, 4 - West, 5 - North West )
  Image - маска имен файлов GIF- и BMP-изображений
  Примеры:
    ADD_FRAMES,0,C:\IMAGES\Image1.GIF
    ADD_FRAMES,1,IMAGES\*.GIF
    ADD_FRAMES,2,Image??.GIF

MOVE_FRAME - переупорядочивание фреймов
MOVE_FRAME,Frames,CurFrame,NewFrame
  Frames - индекс направления ( 0 - North East, 1 - East,  2 - South East, 3 - South West, 4 - West, 5 - North West )
  CurFrame - текущий индекс фрейма в заданном направлении  ( 0, 1, ... FramesCount-1  или  -1, -2, ... -FramesCount )
  NewFrame - новый индекс фрейма в заданном направлении  ( 0, 1, ... FramesCount-1  или  -1, -2, ... -FramesCount )
  Примеры:
    MOVE_FRAME,0,1,2
    MOVE_FRAME,0,-1,0

REMOVE_FRAMES - удаление фреймов
REMOVE_FRAMES,[Frames],[Frame1],[Frame2]
  Frames - индекс направления ( 0 - North East, 1 - East,  2 - South East, 3 - South West, 4 - West, 5 - North West )
  Frame1 - индекс удаляемого фрейма в заданном направлении  ( 0, 1, ... FramesCount-1  или  -1, -2, ... -FramesCount )
  Frame2 - индекс фрейма, по который надо удалить, начиная с Frame1 в заданном направлении  ( 0, 1, ... FramesCount-1  или  -1, -2, ... -FramesCount )
  Примеры:
    REMOVE_FRAMES,0,0,-2
    REMOVE_FRAMES,1,2
    REMOVE_FRAMES,2
    REMOVE_FRAMES

LOAD_PALETTE - загрузка палитры
LOAD_PALETTE,Palette
  Palette - имя файла палитры
  Примеры:
    LOAD_PALETTE,C:\TEMP\Default.act
    LOAD_PALETTE,color.pal

SET_SAMPLE_COLOR - установка образца цвета 
SET_SAMPLE_COLOR,SampleColor
  SampleColor - RGB-цвет
  Примеры:
    SET_SAMPLE_COLOR,0xFF0000
    SET_SAMPLE_COLOR,255

SET_OPTION - установка опции
SET_OPTION,Option,Value
  Option - идентификатор опции ( TRANSP_MODE, AUTOCUTTING, DITHER_METHOD, ACTION_FRAME, FPS )
  Value - значение опции
    - для 'TRANS_MODE'  0 - Auto, 1 - Palette и 3 - Sample
    - для 'AUTOCUTTING'  0 - Нет, 1 - Да
    - для 'DITHER_METHOD'  0 - Default, 1 - Advanced и 2 - Fallout
    - для 'ACTION_FRAME' положительное целое число 0, 1, .... FramesCount - 1
    - для 'FPS' положительное целое число
  Примеры:
    SET_OPTION,TRANSP_MODE,1
    SET_OPTION,AUTOCUTTING,0
    SET_OPTION,DITHER_METHOD,2
    SET_OPTION,ACTION_FRAME,6
    SET_OPTION,FPS,18

NEW - инициализация
NEW 

OPEN - открыть проект или FRM-файл
OPEN,Source,[Palette]
  Source - имя FPR- или FR(FRM,FR0-FR5)-файла
  Palette - имя файла палитры
  Примеры
    OPEN,C:\TEMP\MyFrm.frm,C:\TEMP\color.pal
    OPEN,C:\TEMP\MyProject.fpr

RUN_SCRIPT - выполнить скрипт
RUN_SCRIPT,Script
  Script - имя файла скрипта
  Примеры:
    RUN_SCRIPT,C:\Temp\MyScript.fas

SAVE - сохранить проект или FRM-файл
SAVE,[Dest],[Format]
  Dest - маска имени файла сохранения
  Format – формат сохранения фрэймов (BMP, GIF или PNG)
  Примеры:
    SAVE,C:\TEMP\MyFrm.frm,PNG
    SAVE,C:\TEMP\MyFrm.frm
    SAVE,C:\TEMP\??????ab.fpr
    SAVE
